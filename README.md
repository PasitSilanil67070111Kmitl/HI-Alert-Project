## จุดประสงค์

เพื่อป้องกันและลดความเสี่ยงการเกิด **Heat stroke** สำหรับผู้ที่ทำงานหรือมีกิจกรรมอยู่กลางแจ้ง โดยระบบจะวัดอุณหภูมิและความชื้นสัมพัทธ์ คำนวณเป็นดัชนีความร้อน แล้วแจ้งเตือนพร้อมคำแนะนำผ่าน **Telegram และ Website** และแสดงผลบนบอร์ด (LCD) เมื่ออยู่ใกล้

---

## รายละเอียดระบบ

* **Input:** อุณหภูมิ (Temperature, °C), ความชื้นสัมพัทธ์ (Relative Humidity, %)
* **Output:** ดัชนีความร้อน (Heat Index), ระดับเตือน (เช่น ปกติ, ระวัง, เสี่ยง, อันตราย), ข้อความคำแนะนำ
